Unknown Mother-Goose（不为人知的鹅妈妈）

  • 语言：日语
  • 词 : wowaka（1987年11月4日†2019年4月5日）
  • 曲 : wowaka（1987年11月4日†2019年4月5日）
  • 类型：V家
  • 歌姬：初音ミク
  • 推荐理由：
    这首歌我听了蛮久的，我网易云主页显示的是我听了这首歌100多次（我真的很少听同一首歌听得完完整整超过30次）。第一次听到这首歌的时候，是听up主进击的冰糖唱的，当时真的被迷住了，感觉超级好听，并没有细品歌词的意思。直到有一段时间，我认真看了看歌词，才发现wowaka写这首歌时的心境。现実逃避P的名字，我这才理解它真正的含义。这首歌的歌词，是拥有那样的深意。

他用了6年才把自己曾被伤害的温柔本心拼起来,而那颗心也不再是简单的黑白分明，它在复杂灰暗的现实世界中学会了游刃有余，看似混乱，实则对称有序，这个人不再“现实逃避”，而是温柔地向这个世界传达“爱”的主题。
——知乎用户“百川东到海”

wowaka，一路走好
——2021.02.23

一梦贪欢

  • 语言：中文
  • 词 : 苍苍
  • 曲 : 砖厂浪人
  • 类型：V家
  • 歌姬：洛天依 / 徵羽摩柯
  • 推荐理由：
    这首歌有一种莫名的对人的吸引感。当前奏响起就令人欲罢不能。这首歌用一种略为失真的曲调描绘了一幅拥有色彩的场景。说句实话，我的《夏·思·梦》的灵感有一部分也来自这里。我非常喜欢砖厂浪人写的歌，不知为何有一种亲切的感觉（不知道怎么想的2333）。偶然发现，这首歌开头两句出自苏轼的《行香子·过七里濑》：

一叶舟轻，双桨鸿惊。水天清、影湛波平。鱼翻藻鉴，鹭点烟汀。过沙溪急，霜溪冷，月溪明。
重重似画，曲曲如屏。算当年、虚老严陵。君臣一梦，今古空名。但远山长，云山乱，晓山青。
——苏轼《行香子·过七里濑》

我查了一下这首词的赏析：

此词是苏轼在宋神宗熙宁六年二月（时任杭州通判）的一个清晨乘轻舟经过浙江境内著名风景区富春江上的七里濑以后写下的。词中在对大自然美景的赞叹中，寄寓了因缘自适、看透名利、归真返朴的人生态度，发出了人生如梦的浩叹。
——百度汉语

苍苍把这几句压缩了，但是我看来不失色彩。当然如果脑补大戏的话也是能脑补出来的23333
吸引我的地方还有这首歌的PV，真的超级喜欢！大家可以到原视频应援哦！！！
——2021.02.23
